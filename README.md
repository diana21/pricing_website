# Simple Pricing Plan Website

### This simple pricing plan website is the website that contain :

1. Navigation Bar
2. Pricing Plan
3. Customer Review / Say
4. FAQ (Frequently Asked Question)
5. Get Started Button
6. Footer

### Tools I Used :

1. HTML
2. CSS (Boostrap Framework)
3. JavaScript
4. IDE (VsCode)
5. Design (Figma)
